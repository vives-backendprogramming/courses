### JPA
Oplossing opdracht les JPA en Spring Data JPA

- Start de applicatie op
- Open een browser en navigeer naar http://localhost:8080/h2-console
- Log in met de credentials 
    - Driver class: org.h2.Driver
    - JDBC URL: jdbc:h2:mem:testdb
    - UserName: sa
    - Password:
- Controleer of alle tabellen correct werden aangemaakt
    - tabelnaam
    - kolommen, kolomnaam, kolomdatatypes
    - verplichte kolommen
    - pk's
    - fk's + fk-kolommen
- Controleer of de data is toegevoegd in de tabellen door `CommandLineRunnerAtStartup` na opstarten van de applicatie
